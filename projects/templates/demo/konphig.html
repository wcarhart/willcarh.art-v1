Konphig is my collection of configuration files that I've developed over the years. There are files for:

  • bash functions, aliases, and variables

  • vim configurations

  • git configurations

  • tmux configurations

  • gpg configurations

  • PyPI configurations

Favorites

Since there are over 80 bash functions in Konphig now, I'll just show you my favorites.

up and down - move up and down the file tree

$ pwd
/home/projects/project1/src/
$ up 2
/home/projects/
$ down
/home/projects/project1/
$ down
/home/projects/project1/src/

adda - add a permanent bash alias on the fly

$ adda e echo
$ e 'Hello, world!'
Hello, world!

makef - create a permanent bash function from commands in the history on the fly

$ clear
$ ls
file0.txt
file1.txt
$ makef 2 cls
$ cat ~/.bashrc
cls() {
  clear
  ls
}

snag - move the most recently downloaded file to the current directory

$ ls
$ ls -t ~/Downloads | head -1
new_file.txt
$ snag
$ ls
new_file.txt

what - show what filetypes are in a directory

$ what
.txt
.md
.py
.pyc

getlocation - get your current location (based on your public IP address)

$ getlocation
IP: 105.38.27.128
Location: San Francisco, CA, USA

duf - show disk usage by filetype

$ duf py md
py: 30K
md: 4.7K

rickroll - do you really need an explanation?

$ rickroll       # guess you'll have to try this one yourself ;)
