Despicable is a lightweight framework written in Python for multithreading shell commands

Usage

Despicable can take commands from either the command line:

python despicable.py -c "cmd0" "cmd1" "cmd2"

Or from a file:

python despicable.py -f command_list.txt

Despicable then draws a nice UI on the terminal:

$ python despicable.py -f command_list.txt
[>>>>>>>>>>>>>>>               ] 50% (Processing)
 \ 2 tasks remaining

Example

Suppose you wanted to parallelize the following tasks:

cp /source/largefile.csv /destination/
bzip2 -z data.txt
scp big_file.json username@remotehost.edu:/some/remote/directory
aws s3 cp /tmp/foo/ s3://bucket/ --recursive --exclude "*" --include "*.jpg"

If these commands were in the Despicablefile (default file for commands), you could run it like so

$ python despicable.py
[>>>>>>>>>>>>>>>               ] 50% (Processing)
 \ 2 tasks remaining
