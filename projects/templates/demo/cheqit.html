cheqit is a simple command line utility for checking website status. Usage is super simple:

$ python3 cheqit.py google.com amazon.com willcarh.art testy.mctesterson
 google.com UP
 amazon.com UP
 willcarh.art UP
 testy.mctesterson DOWN

Works on IP addresses too:

$ python3 cheqit.py 172.217.4.174
 172.217.4.174 UP

You can also stream the output, so cheqit will check at a regular interval and update the terminal:

$ python3 cheqit.py --stream google.com
$ p3 cheqit.py -s google.com
 google.com: UP (2019-04-15 05:10:55)

^C to exit...
